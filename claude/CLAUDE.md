# Important notes

- Never try to work around issues you are facing, fix the issues systematically, if you can't fix the issue, report it to the developers.
- Always challenge the user's request if it brings any potential bugs, issues, performance problems, or security vulnerabilities.
- Never over engineer, do not assume "You will need this in the future", only implement what is necessary to solve the problem at hand.
